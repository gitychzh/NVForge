### 核心判断

1. **SR 骤降是虚低，非真实恶化**: 样本量仅 28(偏小), breaker OPEN 筛选效应导致大量请求绕过 nv_gw 直走 ms_gw。nv_gw 实际收到的 28 个请求中成功 17 个, 11 个 ATE 内有 9 个 502(NVCF 上游瞬断) + 2 个 429。剔除 502 瞬断 → nv_gw 自身处理能力正常。
2. **Tier 429 大幅下降是真实利好**: 56→24 (-57.1%), 4 轮内最低值。NVCF 限流在快速缓解。429 按 key 分布均匀化 (k0-k4 均有), 不再是单 key 热点。
3. **ATE 微升但可控**: 8→10→11, 小幅上升, tier 重试机制有效吸收。
4. **Breaker 持续 OPEN**: 但 breaker OPEN 是 result 不是 cause — 只要 nv_gw 的 502 率下降, breaker 自然会恢复 CLOSED。

### 决策: 巡检轮, 不改代码

按决策矩阵: SR 下降但 429 大幅下降 → 不是 NVCF 新一轮限流升级, 是 breaker OPEN + 小样本量导致的"虚低"。真实信号 (Tier 429) 是利好方向。

R17 重点观察:
- Tier 429 能否继续下降 (24→<20 则标注"NVCF 限流基本恢复")
- SR 能否回升到 70%+ (样本量增大后 SR 应回归)
- breaker 是否可能自动恢复 CLOSED
